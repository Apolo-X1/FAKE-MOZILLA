Cuadro comparativo

Lenguaje Significado Uso Características
HTML HyperText Markup Language Estructurar el contenido de una página web Utiliza etiquetas para definir el contenido, no es un lenguaje de programación.
CSS Cascading Style Sheets Estilo visual de una página web Permite definir el estilo visual de una página web, separa la presentación del contenido.
JavaScript Lenguaje de programación Agregar interactividad a una página web Se utiliza para agregar interactividad, dinamismo y funcionalidad a una página web, se ejecuta en el lado del cliente.
PHP Hypertext Preprocessor Lenguaje de programación para el lado del servidor Se utiliza para procesar y generar contenido dinámico en el lado del servidor, puede interactuar con bases de datos y otras tecnologías del lado del servidor.
